API Rest do site da Beit HaSefer, que tem por finalidade ensinar sobre a Torá e a Kabbalah. Projeto em desenvolvimento.

### Tecnologias utilizadas
- TypeScript
- Node.js
- NestJS
- MySQL
- TypeORM
- Jest
