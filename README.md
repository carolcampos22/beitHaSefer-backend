# Beit HaSefer

Repositório backend do projeto Beit HaSefer, um site voltado ao ensino da Torá e da Kaballah.
Projeto em desenvolvimento.

### Tecnologias utilizadas
- TypeScript
- Node.js
- NestJS
- MySQL
- TypeORM
- Jest
